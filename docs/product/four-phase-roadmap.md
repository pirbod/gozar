# Four-Phase Roadmap

Gozar/Gorz is closed as a four-phase controlled prototype roadmap. Phase 4 is the final roadmap phase.

## Phase 1: Local Profile Lifecycle

- device registration
- signed encrypted short-lived demo profiles
- deterministic policy selection
- TTL validation
- revocation
- redacted audit export
- safety pause
- local-only backend

## Phase 2: Android VpnService Prototype

- Android app
- VpnService permission
- Profile API integration
- signature verification
- profile decryption
- route validation
- local TUN interface
- packet counting and dropping
- no public forwarding

## Phase 3: Clickable Android Product Experience

- onboarding
- home
- connect flow
- session dashboard
- confidence
- route policy
- diagnostics
- evidence
- safety pause
- audit
- settings
- offline demo
- emulator smoke test

## Phase 4: Controlled Release Readiness

- hardening
- screenshots
- validation report
- emulator evidence
- threat model
- privacy review
- release artifact manifest
- production readiness report
- final documentation
- release blocker checklist

## Closure Statement

There is no Phase 5 in this roadmap. Further work after Phase 4 is post-roadmap maintenance, independent review, controlled pilot governance, or future production expansion.
