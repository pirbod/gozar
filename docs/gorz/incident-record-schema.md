# Gorz Incident Record Schema

Gorz incident records are redacted JSON documents generated from simulated local evidence. They are intended for review, not automatic upload.

## Fields

- `incidentId`: Locally generated record identifier.
- `appName`: Always `Gorz`.
- `generatedAt`: ISO timestamp for record creation.
- `timeWindow`: Start and end timestamps for the summarized interval.
- `scenario`: Human-readable simulated scenario name.
- `classification`: One of `confirmed_external_access_likely`, `probable_external_access`, `selective_blocking_possible`, `domestic_only_possible`, `general_outage_possible`, `no_proof_of_external_access`, or `inconclusive`.
- `confidenceScore`: Number from 0 to 1.
- `layerScores`: DNS, transport, TLS/QUIC, application delivery, externality, corroboration, and safety penalty scores.
- `evidenceSummary`: Redacted human-readable summary.
- `messageDeliveryState`: Draft, queued, encrypting, sent, delivered, delayed, blocked likely, or failed.
- `safetyControls`: Local safety settings at generation time.
- `redactions`: Explicit list of excluded sensitive data.
- `notes`: Additional safe notes.

## Never Include

- Real IP address
- Phone number
- Exact GPS location
- Real contact identity
- Message plaintext
- Raw message content
- Device identifier

## Example

```json
{
  "incidentId": "incident_demo_001",
  "appName": "Gorz",
  "generatedAt": "2026-01-01T00:10:00.000Z",
  "timeWindow": {
    "startedAt": "2026-01-01T00:00:00.000Z",
    "endedAt": "2026-01-01T00:10:00.000Z"
  },
  "scenario": "Domestic-only connectivity",
  "classification": "domestic_only_possible",
  "confidenceScore": 0.54,
  "layerScores": {
    "dnsScore": 0.85,
    "transportScore": 0.75,
    "tlsScore": 0.7,
    "applicationDeliveryScore": 0.6,
    "externalityScore": 0.18,
    "corroborationScore": 0.45,
    "safetyPenalty": 0.04
  },
  "evidenceSummary": [
    "Mandatory layers are partly available, but external delivery is not confirmed.",
    "Evidence is generated by a local mock simulator for this proof of concept.",
    "Encrypted message content and contact identity are intentionally excluded."
  ],
  "messageDeliveryState": "delayed",
  "safetyControls": {
    "safetyModeEnabled": true,
    "localDiagnosticsOnly": true,
    "disableAutomaticDiagnosticUpload": true,
    "delayIncidentExport": true,
    "hideAdvancedTechnicalDetails": true,
    "reduceDiagnosticFrequency": true,
    "emergencyPause": false
  },
  "redactions": [
    "No real IP address",
    "No phone number",
    "No exact GPS location",
    "No real contact identity",
    "No message plaintext",
    "No raw message content",
    "No device identifier"
  ],
  "notes": ["Simulated PoC data only."]
}
```
